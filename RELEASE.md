# vid-lab v0.1.0 — 23 июня 2026

Первый стабильный релиз. Core-функциональность работает, архитектура вычищена.

## Что внутри

- ✅ Скачивание YouTube / Instagram / TikTok (yt-dlp, 720p)
- ✅ S3 для файлов >42 MiB через Beget (presigned URL)
- ✅ Красивые ссылки через agent.rusinov-s.ru (nginx + FastAPI redirect)
- ✅ Дата в именах файлов на S3 (vid-lab/YYYY-MM-DD/)
- ✅ Лимит 5/день для free-пользователей
- ✅ Регистрация пользователей (база для подписок)
- ✅ SQLite с правильным PRIMARY KEY (без ошибок ON CONFLICT)
- ✅ systemd автозапуск (vid-lab.service + vid-lab-dl.service)
- ✅ Автоочистка файлов (cron vid-lab-cleanup, каждый час)
- ✅ Автоочистка токенов (cron vid-lab-links-cleanup, каждый день в 3:00)
- ✅ Перекрёстное тестирование (12 сценариев)

## Изменения относительно предыдущего состояния

Исправлено 4 бага, добавлена архитектура S3 redirect + датирование файлов.
