## Целевая аудитория

**География:** весь мир, где доступен Telegram. Россия — стартовый рынок, но англоязычный интерфейс + мультиязычность открыты для любой страны.

**Приоритет сегментов (продажи):**

1. **B2B через К-Сервис** — существующая клиентская база (ретейл, HoReCa, производство). Тариф Agency 2 490 ₽/мес. Канал продаж уже есть, ничего дорабатывать не нужно.
2. **Родители / путешественники** — скачать контент перед поездкой. Не по деньгам, а по стратегии: огромный объём, сарафанное радио через мамские чаты. Максимально простой UX — 360p сразу в Telegram, одна кнопка. Конверсия 1-2% в платных.
3. **Контент-мейкеры / блогеры / монтажёры** — когда появится 1080p и выбор качества по умолчанию.
4. **SMM-агентства / маркетинговые агентства** — когда появится API и пакетная загрузка. Самый платёжеспособный, но самый требовательный сегмент.

Подробный анализ сегментов с болями и ценами: docs/audience-segments.md

## Планируется (v0.2 — качество и умная маршрутизация)

2. **Подсказка при большом файле** — если файл >42 MiB для выбранного качества, бот предлагает перекачать в меньшем разрешении (прямо в Telegram)
3. **Умная маршрутизация** — если файл ≤42 MiB → сразу в Telegram, если >42 MiB → S3
4. **1080p только для Pro** — подготовка схемы к подпискам (B2B-сегмент)
5. **X-Accel-Redirect вместо 307** — браузер не покидает наш домен
